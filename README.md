# Student Performance Prediction Using Machine Learning

This project predicts a student's final exam score based on:
- Study Hours
- Attendance Percentage
- Previous Exam Score

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Objective
To help teachers and institutions predict student performance early.

## Output
The model predicts the final score and shows a visualization between study hours and final score.
